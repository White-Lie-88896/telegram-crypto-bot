## 快速开始

### 1. 安装依赖

```bash
# 安装aiohttp
pip install aiohttp
```

### 2. 启动服务器

```bash
# 运行Web服务器
python start_web.py

# 或者直接运行
python web/server.py
```

### 3. 访问后台

打开浏览器访问：
```
http://localhost:8080
```

## API接口

### 统计数据
```
GET /api/stats
```
返回活跃任务数、预警次数等统计数据

### 实时价格
```
GET /api/prices
```
返回多个币种的实时价格和涨跌幅

### 监控任务
```
GET /api/tasks
```
返回所有活跃的监控任务列表

### 预警历史
```
GET /api/alerts?limit=50
```
返回最近的预警记录

### 系统状态
```
GET /api/system
```
返回系统运行状态和健康指标

### 用户列表
```
GET /api/users
```
返回所有注册用户

## 配置说明

### 服务器配置
在 `web/server.py` 中修改：

```python
# 主机地址
host = '0.0.0.0'  # 监听所有接口

# 端口号
port = 8080       # 默认端口
```

### 前端配置
在 `web/static/js/app.js` 中修改：

```javascript
// API基础URL
const API_BASE = '/api';

// 更新间隔（毫秒）
const UPDATE_INTERVAL = 30000;  // 30秒
```

## 开发指南

### 添加新页面
1. 在 `templates/` 创建HTML文件
2. 在 `server.py` 添加路由
3. 更新导航菜单

### 添加新API
1. 在 `WebDashboardAPI` 类中添加方法
2. 在 `setup_routes()` 注册路由
3. 在前端 `app.js` 调用API

### 自定义样式
编辑 `static/css/style.css`：
- 修改CSS变量自定义主题
- 添加新的样式类
- 调整响应式断点
